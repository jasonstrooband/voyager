## Implementation

### Dependencies
Dependencies are loaded from urls

Currently using:
* jQuery
* Bootstrap
* Bootply Slate Theme
* Popper
* Font Awesome

### Installation
Requires the use of a web server like [Wamp](http://www.wampserver.com/en/) to use.
